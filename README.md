# createEXE
Program that creates an EXE file of a program written in python.

Process:
1. User selects the directory of the python file for which the EXE file is to be created.
2. User starts processing.
3. Folder 'exe' is created in the working directory, and it is than set as the new working directory.
4. Command is executed in the CMD.
5. Processing is finalized.
6. Message box with a notification on successful processing is displayed.
7. User can open the file explorer with the location of the new EXE file.
